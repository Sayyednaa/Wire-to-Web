Setup Windows Agent

To link physical USB or parallel printers to your account, download and run our silent Windows Agent installer on the host computer.

📥 Download Agent 💻 View Source on GitHub
Setup Instructions
  1. Download and run the install_agent.exe standalone installer on the Windows PC connected to your printer hardware.
  2. When prompted, enter the Server URL:
    {{ request.scheme }}://{{ request.get_host }}
  3. Sign in using your account username and password.
  4. The installer will automatically register your active printers, install binaries to Local AppData, set the Registry autostart key, and launch the background printing daemon silently (running windowless with no command prompt window).
